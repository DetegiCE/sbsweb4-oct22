조건문, 선택문, 반복문
- 조건 : 조건에 따라서 특정 코드를 실행하고 말고
- 선택 : 값에 따라서 값이 있으면 선택해서 실행
- 반복 : 코드를 특정 횟수 (특정 조건에 해당하면) 반복

조건문 : 조건식이 참인지 거짓인지에 따라서 제어
if 문: 조건이 만족하면 if문을 실행한다

if ( 조건식 ) {
  코드
} 

else 문: if 조건식이 해당하지 않을 경우 실행하는 구문

if ( 조건식 ) {
  조건이 참일 때 실행되는 코드
}
else {
  조건이 거짓일 때 실행되는 코드
}

반복문 -> 코드를 원하는 횟수만큼 반복하여 실행할 수 있음
while 문 - do while 문 - for 문

반복문을 위한 변수를 작성할 때에는 i, j, k, ....
i = iteration 의 줄임말 (반복하다)
it, iter

while 문 : 조건식을 만족할 동안 계속 실행되는 코드

while( 조건식 ) {
  코드
}

안녕하세요 1 ~ 30번까지 출력
짝수번째 (2, 4, 6, 8, ...)은 파란색으로
홀수번째 (1, 3, 5, 7, ...)은 빨간색으로

do while 문 - 적어도 1번 조건이 맞지 않더라도 실행 한다

do {
  코드
} while( 조건식 );


******* for 문 *******
for문도 조건식을 만족할 때까지 특정 코드를 반복 실행 

for ( 초기식 ; 조건식 ; 변화식 ) {
  코드
} 

시작 -> 초기식 -> 조건식(참) -> 코드 -> 변화식 -> 조건식(...)
                         조건식 (거짓) -> X

for 문을 이용해서 1~50 사이의 수에 대해서
5와 7의 공배수면 초록색
5의 배수면 빨간색
7의 배수면 파란색
셋 다 아니면 검정색
