함수이름(매개변수에 따른 값);

var idArray = ["sbs1019", "hello1234", "james111"]
var pwArray = ["1234", "12345678", "james000"]
위와 같이 계정 정보가 저장되어 있을 때,
prompt를 이용해서 id와 비밀번호를 입력받고,
함수에 입력받은 값을 매개변수로 넘겨준 뒤,
함수 내부에서 로그인 체크
id 가 없음 -> "존재하지 않는 아이디입니다."
pw 가 틀림 -> "비밀번호가 틀렸습니다."
둘다 맞음 -> id + "님 환영합니다."

body에 버튼 두 개를 추가
1. 로그인 버튼 -> 지금 했던 prompt부터 로그인 체크까지
2. 회원가입 버튼 -> 회원가입 버튼을 눌렀을 때 회원가입 함수가 실행되는데, 회원가입 함수는 prompt를 이용하여 id와 비번을 각각 한번 씩 입력해서 그 값들을 idArray, pwArray 에 추가
