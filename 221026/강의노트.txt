객체 편집 메소드 (함수)
같은 위계에 추가하는 요소
$(요소).before(새 요소) -> 요소의 전 위치에 새 요소를 추가한다
$(새 요소).insertBefore(요소)
$(요소).after(새 요소) -> 요소의 뒤 위치에 새 요소를 추가한다
$(새 요소).insertAfter(요소)

자식을 추가하기
$(요소).append(새 요소) -> 요소의 자식의 맨 뒤에 새 요소를 추가한다
$(새 요소).appendTo(요소)
$(요소).prepend(새 요소) -> 요소의 자식의 맨 앞에 새 요소를 추가한다
$(새 요소).prependTo(요소)

$(요소).clone([false]) -> HTML, CSS 요소만 복제
$(요소).clone(true) -> + 이벤트(동작)까지 복제
$(요소).empty() -> 요소의 모든 하위 요소를 삭제
$(요소).remove() -> 해당 요소를 삭제
