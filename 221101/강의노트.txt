이벤트 코드 작성
.click -> 직접 이벤트 이름 작성하는 경우
.on("click", -> on 안에 이벤트 이름을 작성하는 경우
.off("click", -> on으로 줬던 이벤트제거


로딩 이벤트 -> 사용자가 브라우저에 HTML을 요청할 때
- ready(): 준비한다
- load(): 로딩완료

마우스 이벤트
- click() : 클릭 했을 때
- dblclick() : 더블클릭 했을 때
- mouseover() : 특정 요소에 마우스 포인터를 올릴 때
- mouseout() : 특정 요소에서 마우스 포인터가 벗어날 때
- hover() : 특정 요소에 마우스 포인터가 올라갈때와 벗어날 때 -> 2개의 함수를 사용 (1번째 함수는 올라갈 때, 2번째 함수는 내려갈 때)
- mouseenter() : 특정 요소의 경계 범위에 마우스 포인터가 들어가는 경우
- mouseleave() : 특정 요소의 경계 범위에서 마우스 포인터가 벗어나는 경우
- mousemove() : 마우스가 움직였을 때 -> 마우스를 움직일 때마다 움직인 것에 대한 이벤트 객체를 생성함 -> function(e) : e가 이벤트 객체다

이벤트 객체
.on(이벤트 함수, function(e) {});
- 마우스 이벤트의 이벤트 객체
 마우스 포인터의 X, Y 좌표값
* clientX, clientY -> (스크롤 무시)
* pageX, pageY -> (스크롤 무시X)
* screenX, screenY -> (모니터 기준)
* layerX , layerY -> (특정 요소 기준)
* button -> 마우스 클릭 했을 때 어떤 마우스로 눌렀는가? (왼쪽: 0, 오른쪽: 2, 휠버튼: 1)

- scroll() : 스크롤 할 때 발생하는 이벤트

포커스 이벤트 -> a 태그의 탭으로의 선택, input을 클릭한 경우, 마우스로 가져다 대서 활성화 된경우
마우스가 없어도 키보드만을 이용해서 요소에 접근할 수 있게끔 해주는 것 (키보드 접근성)
- focus() : 해당 요소에 포커스가 발생할 경우
- blur() : 포커스가 다른 요소로 이동하는 경우
- focusin() : 부모의 요소 중에 입력 자식 요소로 포커스가 이동할 경우
- focusout() : 입력 자식 요소에서 포커스가 벗어날 경우
