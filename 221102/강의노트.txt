키보드 이벤트
keydown() -> 눌렀을 때 (키보드의 모든 키)
keypress() -> 눌렀을 때 (키보드에서 기능 키를 제외한 모든 키) (F1~F12, Prt Sc, Lock, Tab, Caps Lock... -> X)
keyup() -> 뗄 때 

키보드 이벤트 객체
keyCode -> 키보드의 아스키 코드 (키보드에 있는 키들을 숫자로 치환해 놓은 것)
-> '0' 48, 'A' 65, 'a' 97
altKey
ctrlKey
shiftKey

키보드 입력을 이용해서 박스를 이동시켜봅시다.
처음에 100x100 크기의 박스가 (500, 500) 위치에 있음
위 화살표(38) -> y 감소
아래 화살표(40) -> y 증가
왼쪽 화살표(37) -> x 감소
오른쪽 화살표(39) -> x 증가
switch-case 문 사용


그룹 이벤트 등록
-> 한 번에 2개 이상의 이벤트를 등록할 수 있음
ex) 마우스 포인터를 올렸을 때, 포커스 생성되었을 때

on() -> 이벤트를 발생시키는 대상에 2개 이상의 이벤트를 등록할 때 

HTML로딩이 완료된 후, 그 후에 새롭게 생성된 객체에 대해서도 이미 등록한 이벤트가 적용이 됨
-> 라이브 이벤트 등록 방식 사용해야 함
$(document).on(이벤트 종류, 이벤트 대상, 함수);
변화사항

delegate() -> 선택한 요소의 하위 요소에 이벤트를 등록, 이벤트를 등록한 후에도 그 이후에 생성된 요소에도 이벤트 적용이 됨 (동적 이벤트 생성)
delegate 함수는 on 함수와 순서가 다름
on(이벤트 종류, 대상, 함수)
delegate(대상, 이벤트 종류, 함수)

one() -> 이벤트가 1회 발생하면 삭제 됨 (일회성 이벤트) (동적 이벤트 생성 가능)
one 함수의 작성 순서는
one(이벤트 종류, 대상, 함수)


이벤트 삭제
on 이벤트를 삭제할 때 off
delegate 이벤트를 삭제할 떄 undelegate
$(요소).off(이벤트 종류)
$(요소).undelegate(이벤트 종류)


효과, 애니메이션

효과 메소드
toggle() -> hide() & show()
fadeToggle() -> fadeIn() & fadeOut()
slideToggle() -> slideUp() & slideDown()
fadeTo() -> 지정된 투명도로 바뀐다

$(요소).효과메소드(소요시간, 가속도, 콜백함수);
소요시간: slow, normal, fast, 1000(1초), 500(0.5초) 등
가속도: swing(시작과 끝은 느리게, 중간은 빠르게), linear(일정한 속도)
콜백 함수: 효과가 종료된 후에 실행되는 함수

animate() 메소드: 선택한 요소에 대해서 모션 효과를 부여할 수 있음 (css 효과)
$(요소).animate({스타일}, 시간, 가속도, 콜백함수)
